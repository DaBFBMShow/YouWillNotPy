# You Will Not Py
A text-based short game about trying to play a game written entirely in python!
## Description
You are a user trying to access a program while the programmer is trying to stop you! Less exciting than it sounds!
This game is under development. Expect PLENTY of bugs
## Inpirations
There Is No Game and The Stanley Parable is the two most notable games that comes to mind for me.
